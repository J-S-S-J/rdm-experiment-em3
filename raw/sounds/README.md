# Sound files for pre-exposure block

Place your two audio files in this folder with these exact names:

- `control_pink_noise.wav` (used when Task Type = control)
- `binaural_sound.wav` (used when Task Type = binaural)

The experiment will play the selected file for 20 minutes before the RDM task starts.
If a file is shorter than 20 minutes, it will automatically loop.
